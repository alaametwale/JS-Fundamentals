# JavaScript Fundamentals

This project is part of the ALX Front-End Web Development program.

## Description
This repository contains solutions for the JavaScript Fundamentals project.
The goal of this project is to learn the basics of JavaScript including:
- Variables and constants
- Data types
- Control structures
- Functions
- Loops
- Command-line arguments

## Requirements
- All files are written in JavaScript
- All files are executable
- The first line of each file is `#!/usr/bin/node`
- No usage of `var`
- Output is printed using `console.log()`
